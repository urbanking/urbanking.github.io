Overview

Future study pages will present the research question, method, data scope, evaluation design, and limitations in one readable place. Paper, code, data, and visualization links will appear only when those resources are genuinely public.

Planned structure

Each page will separate individual-choice diagnostics from aggregate distribution results and clearly identify whether a figure is observed, model-implied, calibrated, or simulated. Reproducibility notes will document data boundaries and evaluation assumptions.

Interactive visualization demo

The embedded chart is a standalone HTML demonstration hosted directly on GitHub Pages. Use the year controls to confirm that the interface works.
